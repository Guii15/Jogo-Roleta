# 🎡 Jogo da Roleta Interativa

Um projeto desenvolvido para fins de estudo e prática de lógica de programação, integrando tecnologias Web e Backend.

## 🛠️ Tecnologias
* **Python (Flask):** Servidor backend para processamento de dados.
* **JavaScript:** Animações da roleta e lógica de sorteio no cliente.
* **HTML5 & CSS3:** Interface visual e estilização.

## 💡 O que eu pratiquei aqui:
* Manipulação de **DOM** com JavaScript para criar o movimento da roleta.
* Comunicação entre **Frontend e Backend** (integração do script com o Python).
* Estilização com CSS para criar uma interface lúdica.

---
"Projeto criado para experimentação de algoritmos de sorteio e interatividade."
